# Frontend - Ithaka Assistant

SPA para postulaciones de futuros proyectos en ithaka.

## Descripción

Este proyecto es la interfaz frontend del asistente inteligente de Ithaka, una plataforma que facilita la comunicación y gestión de postulaciones a través de una interfaz de chat conversacional.

## Instalación y configuración

1. Clonar el repositorio
2. Navegar a la carpeta del proyecto:
```bash
cd frontend-ia
```

3. Instalar dependencias:
```bash
npm install
```

## Ejecutar el proyecto

### Modo desarrollo
Ejecutar desde la raíz del repositorio:
```bash
cd frontend-ia
npm run dev
```

El proyecto se abrirá en `http://localhost:5173`

### Build para producción
```bash
npm run build
```
